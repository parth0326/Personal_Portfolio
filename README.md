# Parth Patel — Personal Portfolio

Live site: [parthpatelportfolio.org](https://parthpatelportfolio.org)

Personal portfolio website for Parth Patel, Cybersecurity major with a Deep Learning / AI minor at Miami University, Class of 2027.

---

## Tech Stack

Built from scratch — no frameworks, no libraries.

- **HTML5** — semantic structure
- **CSS3** — custom properties, grid, flexbox, animations
- **JavaScript** — vanilla ES6+
- **Fonts** — JetBrains Mono, Inter (Google Fonts)
- **Hosting** — Netlify
- **Domain** — Netlify DNS

---

## Features

- Dark / Light mode toggle with `localStorage` persistence
- Typed text animation cycling through role descriptions
- Staggered scroll-triggered fade-in animations
- Filterable projects grid by category
- "Currently" section showing active focus areas
- One-click email copy with confirmation flash
- Mobile-responsive with bottom navigation bar
- Subtle grid overlay background on hero section

---

## Local Development

No build step required. Just open `index.html` in a browser or serve with any static file server:

```bash
# Using Python
python -m http.server 8000

# Using Node
npx serve .
```

---

## Deployment

Deployed via [Netlify](https://netlify.com) with continuous deployment from this repository. Every push to `main` automatically redeploys the live site.
